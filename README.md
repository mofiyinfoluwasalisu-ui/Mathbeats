# MathBeats 🎵➗

MathBeats is a fun and interactive educational web project designed to make learning mathematics more engaging through music, quizzes, rewards, and mini-games.

The project combines simple math lessons with gamified learning experiences to help students stay motivated while practicing important math concepts.

---

## ✨ Features

### 🎓 Learn Math Terms
- Simple explanations of math vocabulary
- Examples for each concept
- Interactive learning section
- Audio support using speech synthesis

### 🎮 Mini Quiz Game
- Multiple-choice math questions
- Instant feedback for answers
- Point and streak system
- Level progression system

### 🏆 Rewards & Badges
- Unlock achievement badges
- Track points and highest level
- Reward system to encourage learning consistency

### 🎵 Music Mode
- Converts text and numbers into melodies
- Uses the Web Audio API
- Makes learning more creative and interactive

### 💡 Motivation System
- Random motivational quotes
- Encourages students to keep practicing

## 🚀 How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/mofiyinfoluwasalisu-ui/Mathbeats.git
```

### 2. Open the project folder

```bash
cd Mathbeats
```

### 3. Run the project

Open `index.html` in your browser.

You can also use the VS Code Live Server extension for a better experience.
